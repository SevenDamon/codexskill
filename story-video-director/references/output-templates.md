# Output Templates

Load this reference when producing structured outputs for `$story-video-director`.

## Initial Request Template

When the user wants to process a new lesson text or story, invite or infer this structure:

```text
【原文】
粘贴语文课文 / 故事正文 / 剧情段落 / 短片概念

【用途】
课堂导入 / 课文情境再现 / 辅助理解重点段落 / 朗读配画 / 公开课展示 / 视频号内容 / 电影短片 / 故事预演 / 概念视频 / 广告叙事

【目标风格】
写实电影感 / 儿童绘本感 / 国风水墨 / 动画电影感 / 课本插画感 / 类型片质感 / 写实悬疑 / 温暖生活流

【忠实度】
严格忠于原文 / 基本忠于原文但可视频化改写 / 允许适度艺术化

【受众与限制】
学生观看 / 成人向短片 / 品牌内容 / 无暴力 / 可保留类型片冲突 / 其他限制

【目标视频长度】
总长 30 秒 / 60 秒 / 90 秒 / 只做某一段

【需要输出】
文本诊断 / 一致性资产设定 / 参考图提示词 / 片段拆分 / 导演手记输入卡 / GPT Image 2 图片提示词 / Seedance 视频提示词
```

## Director-Note Input Card

```text
## 片段{编号}｜《{片段标题}》

### 【片段作用】
建立场景 / 引入人物 / 制造冲突 / 推进情绪 / 情绪爆发 / 结果收束

### 【目标时长】
8-12秒，最长不超过15秒

### 【剧情动作】
用1-3句话说明这段视频里发生什么。

### 【场景】
地点、时间、环境元素、天气、光线。

### 【角色】
人物数量、关系、状态。不写复杂外貌。

### 【继承参考图】
角色参考图：
物体参考图：
场景参考图：
统一风格参考图：

### 【一致性锁定】
本段必须保持不变的人物、物体、场景方向、色彩风格和视觉锚点。

### 【允许变化】
本段允许变化的情绪、动作、位置、物体状态、天气、光线进展或镜头运动。

### 【核心情绪】
3-5个关键词。

### 【视觉风格建议】
色调、光线、质感、镜头气质。

### 【整体节奏】
起：开头的画面/情绪状态。
承：动作或关系推进。
转：最小变化、压力或反应。
合：结尾停在什么画面/情绪。

### 【1-2个关键镜头】
只写最重要的镜头。

### 【关键镜头细项】
镜头1：
- 情绪目标：
- 摄影机位置：
- 镜头语言：
- 观众注意点：

镜头2：
- 情绪目标：
- 摄影机位置：
- 镜头语言：
- 观众注意点：

### 【调度关系】
人物位置、摄影机位置、运动方向、空间关系。

### 【摄影指导】
使用短语，例如中近景慢推、低机位压迫、手持逼近、高机位俯压、弧线环绕、长焦压缩。只写核心镜头思路。

### 【表演指导】
眼神、呼吸、姿态、手势、身体张力、情绪释放方式。不画真实表情。

### 【灯光与氛围】
关键环境元素、主光/背光/实景灯、氛围词。

### 【色彩设计】
色彩搭配及其情绪作用。

### 【声音设计】
环境音、音乐、静默、关键台词。

### 【台词】
有就写；没有就写“无台词”。

### 【禁止项】
不希望视频模型生成的东西。

### 【页面文字要求】
图片内文字全部中文；短标签、关键词、碎片化导演批注；不要长段落、表格或密集标题。

### 【导演重点一句话】
这一段最重要的导演提醒。
```

- 不要最终电影画面。

- 不要把整篇课文塞进一张图。

## Seedance Prompt

```text
## A版：标准控制版

【Seedance 主提示词】
根据配套导演手记图，生成一个 {目标时长} 的短视频。场景是{场景}。{角色}。动作顺序：{剧情动作}。整体情绪是{核心情绪}。视觉风格为{视觉风格建议}。

【一致性锁定】
本片段必须继承以下参考资产：{继承参考图}。必须保持：{一致性锁定}。只允许变化：{允许变化}。

【镜头节奏】
开始：建立场景和人物关系。
中段：完成核心动作或情绪变化。
结尾：停在最能服务课堂理解的画面。

【负面限制词】
不要新增无关人物；不要频繁切换场景；不要夸张表演；不要改变原文事实；不要生成复杂外貌细节；不要加入无关道具；不要让动作超过一个核心事件；不要改变人物年龄、体型、服装颜色、关键物体形态、场景方向或统一色彩风格。

【一句话生成重点】
{导演重点一句话}

【图文对齐提醒】
导演手记图负责镜头感、调度关系和视觉重点；本提示词负责动作顺序、时长控制和禁止项。生成时优先保持两者一致。

## B版：极简强约束版

生成 {目标时长} 短视频。只拍一个事件：{剧情动作}。场景固定为{场景}。人物只包括{角色}。情绪：{核心情绪}。继承参考资产：{继承参考图}。必须保持：{一致性锁定}。只允许变化：{允许变化}。按导演手记图的构图和调度执行。禁止新增人物、换场景、过度表演、加入原文没有的情节、改变角色/物体/场景/风格锚点。重点：{导演重点一句话}
```

## Feedback Revision Map

```text
## 问题归类
- 画面好看但不符合课文：提高忠实度，减少艺术化补充。
- 氛围对但动作不对：强化 Seedance 动作顺序。
- 动作对但没有镜头感：强化导演手记图的关键镜头和机位。
- 视频太乱：删次要动作，减少角色，缩短片段目标。
- 人物形象跑偏：减少外貌描述，使用统一角色设定图或占位人物。
- 角色跨段不一致：补强角色设定卡和角色参考图，重复服装/体型/道具锁定词。
- 关键物体变形：补强物体设定卡，锁定尺寸、材质、颜色、状态变化和禁止替换项。
- 场景方向不一致：补充场景俯视图或空间方向参考图，锁定入口、窗户、主体物和摄影机轴线。
- 色彩风格漂移：补充统一视觉风格卡，锁定主色、对比度、光线和镜头质感。

## 修正输出
【应修改的环节】
【原因】
【修正版内容】
【下次试跑重点】
```
